Available Endpoints

  • POST /api/v2/sessions - Create session
  • GET /api/v2/sessions/{id}/state - Get state
  • POST /api/v2/sessions/{id}/edit - Apply edit
  • POST /api/v2/sessions/{id}/undo - Undo
  • POST /api/v2/sessions/{id}/redo - Redo

MVC Principles

  • ✓ Backend holds single source of truth
  • ✓ Frontend is stateless view layer
  • ✓ All operations refresh from backend
  • ✓ Linear undo/redo history
  • ✓ Consistent token abstraction
